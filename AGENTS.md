# AGENTS.md

## Repository Rules

- Use `maintain-gradle-skill` when working in `skills/gradle/**`.
- Do not hard-wrap Markdown prose or list items by column width; keep natural paragraphs and bullets on one line unless Markdown syntax, tables, or code blocks require line breaks.
- Format `.mjs` files with `npx --yes prettier --write <file>`.
- Lint Markdown files from the repository root with `npx --yes markdownlint-cli2 <files> --config .markdownlint-cli2.yaml`.
- Use Conventional Commits for all commit messages. For English subjects, use sentence case after `type(scope):`.
